# Domain 2 Cheatsheet

- SAML = authentication; SCIM = provisioning and deprovisioning.
- EMU is a separate enterprise model, not a toggle on standard GHEC.
- Existing PATs must be authorized for SAML-protected orgs.
- Fine-grained PATs are preferred for narrower scope.
- Team sync requires SAML plus SCIM.
- GHES supports built-in auth, LDAP, CAS, and SAML.
