# Domain 5 Cheatsheet

- GHAS covers code scanning, secret scanning, and dependency review.
- GHAS is free for public GitHub.com repos; private/internal repos need licensing.
- Secret scanning alerts are detective; push protection is preventative.
- Dependabot alerts depend on the dependency graph.
- SARIF is the upload format for third-party code scanning.
- Use security overview and audit log streaming for scale and compliance.
