# GHES Admin Command Reference

- `ghe-config-apply`: Apply management configuration changes.
- `ghe-license`: Upload or inspect GHES licensing.
- `ghe-repl-setup`: Configure a replica for HA.
- `ghe-repl-start`: Start replication.
- `ghe-repl-status`: Check replication status.
- `ghe-repl-promote`: Promote a replica during failover.
- `ghe-maintenance -s`: Enter maintenance mode.
- `ghe-maintenance -u`: Exit maintenance mode.
- `ghe-backup`: Create a backup snapshot from the backup host.
- `ghe-restore`: Restore from a backup snapshot.
